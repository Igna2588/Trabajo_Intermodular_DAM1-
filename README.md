Gestión de Gastos Familiares
Proyecto Intermodular de 1º DAM - Prometeo
Autor: Ignacio Arroyo

¿Qué es este proyecto?

Aplicación de escritorio desarrollada en Java con interfaz gráfica JavaFX 
que permite a una familia registrar, categorizar y controlar sus gastos 
e ingresos mensuales.

¿Qué problema resuelve?

Muchas familias no saben exactamente en qué gastan su dinero cada mes. 
Esta aplicación centraliza todos los movimientos económicos del hogar 
y muestra en tiempo real cuánto se ha ingresado, cuánto se ha gastado 
y cuánto se ha ahorrado.

Funcionalidades

Registrar gastos e ingresos con importe y descripción
Categorizar por tipo con subcategorías reales
Distinguir entre gastos fijos y variables
Ver el balance del mes con ingresos, gastos y ahorros
Los ahorros cambian de color según sean positivos o negativos
Fecha automática al registrar un movimiento
Eliminar movimientos seleccionándolos de la lista

Instalación

1. Instalar Java 17 o superior
2. Instalar XAMPP y arrancar MySQL
3. Ejecutar schema.sql en phpMyAdmin
4. Ejecutar datos.sql para cargar datos de ejemplo
5. Abrir el proyecto en IntelliJ IDEA
6. Configurar VM options con la ruta de JavaFX
7. Ejecutar Main.java
