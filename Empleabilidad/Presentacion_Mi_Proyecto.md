Presentación del Proyecto

¿Qué es el proyecto?

Es una aplicación de escritorio desarrollada en Java con interfaz gráfica 
JavaFX que permite a una familia gestionar sus gastos e ingresos mensuales 
de forma sencilla y visual.

¿Qué problema resuelve?

Muchas familias no saben exactamente en qué gastan su dinero cada mes. 
Esta aplicación centraliza todos los movimientos económicos del hogar 
y muestra en tiempo real cuánto se ha ingresado, cuánto se ha gastado 
y cuánto se ha ahorrado.

¿Para quién está pensado?

Está pensado para cualquier familia que quiera llevar un control de sus 
finanzas sin necesidad de conocimientos técnicos. La interfaz es simple 
e intuitiva.

¿Qué tecnologías utiliza?

Java 24 como lenguaje principal
JavaFX para la interfaz gráfica
FXML para definir la interfaz separada del código
CSS para los estilos visuales
MySQL como base de datos local
JDBC para conectar Java con la base de datos
Git y GitHub para el control de versiones
Maven para gestionar las dependencias

¿Qué he aprendido?

A diseñar una base de datos real desde cero con sus relaciones
A conectar una aplicación Java con una base de datos mediante JDBC
A crear interfaces gráficas con JavaFX y FXML
A separar el código en capas con responsabilidades claras
A separar los estilos visuales en un archivo CSS externo y aplicarlo a la interfaz
A usar Git para controlar versiones y subir código a GitHub
A documentar un proyecto de forma profesional
A trabajar con XML y validarlo con XSD
